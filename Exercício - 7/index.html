Variáveis

Variáveis permitem para nós armazenar uma referência para um valor que será referenciado depois. Quando nós criamos uma nova variável, nós estamos declarando que significa que o nosso código agora reconhece a variável declarada. Variáveis são criadas usando a palavra chave var. Nomes de variáveis não podem começar com número, contendo espaços, ou usar alguma outra palavra que já é reservada para outra coisa. Também não podemos usar qualquer pontuação como: "_" ou "$.
