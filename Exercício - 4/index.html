Operadores em JavaScript

Operadores
+ Adição. Retorna a soma de dois números, se o valor são iguais. Quando adicionamos 2 valores que não são números, então concatenamos ou combinamos os dois valores dentro de uma string.
- Subtração. Retorna o resultado do primeiro número menos o segundo. Podemos também usar o indicativo do valor negativo.
* Multiplicação. Retorna o resultado de dois valores multiplicado juntos.
/ Divisão. Retorna o resultado do primero valor dividido pelo segundo valor.
% Módulo. Retorna a sobra do primeiro valor dividido pelo segundo.
-- Decrementa um valor
++ Incrementa um valor
> Verifica se o primeiro valor é maior do que o segundo valor. E depois retorna true ou false.
< Verifica se o primeiro valor é menor do que o segundo valor. E depois retorna true ou false.
