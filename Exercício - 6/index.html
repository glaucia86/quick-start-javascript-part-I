Tipos de Dados Primitivos em JavaScript

Existe muitos tipos de dados que são usados em JavaScript que servem como blocos básicos para todo programa em Js. E estes são chamados de tipos de dados primitivos e consiste nos seguintes tipos:

  • String
  • Number
  • Boolean
  • Null
  • Undefined
    • * Estes tipos de dados são estranhos!!

Qualquer expressão em JavaScript que for retornar um valor (ex.: 2 + 2 retornará 4) e que terá um tipo. Por exemplo, o resultado de 2 + 2 é 4 e valor 4 é um tipo de dado 'number' (numérico). Agora se você adiciona 2 + "2", o resultado será "22" e como consequencia "22" é do tipo string.

O tipo do valor irá informar qual será o tipo de dado primitivo em JavaScript.
