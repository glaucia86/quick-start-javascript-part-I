Controle de Fluxos

Usando os controles de fluxo nos permite executar somente um determinado código sobre certas condições. 'If/else' são um dos controles de fluxo mais comuns. Ele permite executar o código somente se o if se um determinada condição foi encontrada primeiro. E como consequencia, caso a nossa condição seja encontrada, o código executará e realizará algo no programa. Já o else se uma determinada condição não foi encontrada.
